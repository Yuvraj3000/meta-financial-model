# Meta Platforms Financial Model & DCF Valuation

## Overview

This project is a comprehensive financial model of Meta Platforms developed in Microsoft Excel. The model integrates the Income Statement, Balance Sheet, and Cash Flow Statement to perform company valuation using the Discounted Cash Flow (DCF) method.

The objective of this project was to strengthen practical financial modeling skills by building a dynamic, assumption-driven valuation model similar to those used in investment banking, equity research, and corporate finance.

---

## Features

- Integrated Three-Statement Financial Model
- Revenue Forecasting using operating drivers
- Assumption-driven Forecast Model
- Working Capital Schedule
- Depreciation & Amortization Schedule
- Debt & Equity Schedule
- Free Cash Flow to Firm (FCFF)
- WACC Calculation
- DCF Valuation
- Terminal Value using:
  - Perpetuity Growth Method
  - Exit Multiple Method
- Base, Best, and Worst Case Scenario Analysis
